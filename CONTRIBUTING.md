# Contributing

Contributions are welcome. Keep it simple.

## How to Contribute

1. Fork the repo
2. Create a branch (`git checkout -b fix/your-fix`)
3. Commit your changes (`git commit -m "fix: describe what you fixed"`)
4. Push to your branch (`git push origin fix/your-fix`)
5. Open a Pull Request

## Guidelines

- Keep changes minimal and focused
- Test your script in Discord Desktop DevTools before submitting
- Follow the existing code style â€” no comments unless absolutely necessary
- Use [conventional commits](https://www.conventionalcommits.org/) for commit messages

## Reporting Issues

Open an issue with:
- Discord Desktop version
- What quest type you were trying to complete
- Console error output (if any)
